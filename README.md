## Design

- **Theme:** Warm Lamp Dark UI — dark backgrounds with warm amber/gold accents
- **Fonts:** DM Serif Display (headings), Inter (body), Newsreader (accents), JetBrains Mono (code)
- **Philosophy:** Calm UI — clean, editorial, luxury-adjacent. Not corporate, not noisy.
